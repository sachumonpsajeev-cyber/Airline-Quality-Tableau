## 📊 Dashboard Overview

Interactive Tableau dashboard analysing passenger satisfaction patterns across **129,880 airline passengers**, built using the Airline Quality Ratings dataset from Kaggle.

---

## 📌 KPIs Tracked

| KPI | Value |
|-----|-------|
| Total Passengers | 129,880 |
| Avg Flight Distance | 1,190 miles |
| Avg Passenger Age | 39 years |
| Avg Arrival Delay | 15.09 minutes |
| Avg Departure Delay | 14.71 minutes |

---

## 📈 Visualisations Built

| Chart | Description |
|-------|-------------|
| Donut Chart | Overall satisfaction — 56.55% Satisfied vs 43.45% Neutral/Dissatisfied |
| Bar Chart | Satisfaction by passenger type (Business vs Personal) |
| Column Chart | Satisfaction by travel type (Business · Economy · Economy Plus) |
| Grouped Bar Chart | Passenger info by age group & class |
| Stacked Bar Chart | Passenger choice by flight leg (Long / Medium / Short) |

Interactive filters: **Customer Type · Gender · Type of Travel**

---

## 🧹 Data Quality Steps

- Identified and resolved null values, blank fields, and duplicate records in Excel before connecting to Tableau
- Cross-validated all KPI metrics against raw dataset to confirm accuracy
- Rebuilt Tableau data connection after cleaning to ensure reliable visualisations
- Applied iterative design review — rebuilt charts after discovering mid-project data quality issues
