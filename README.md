# Airline Quality Rating Analysis — Tableau Dashboard

**Course:** Business Intelligence & Data Visualization | MSc Data Science & AI  
**Institution:** TSI University / UWE Bristol | Riga, 2025  
**Tool:** Tableau Desktop · Microsoft Excel

## Dashboard Overview
Interactive Tableau dashboard analysing passenger satisfaction across 129,880 airline passengers.

## KPIs Tracked
- Total Passengers · Avg Flight Distance · Avg Passenger Age
- Avg Arrival Delay · Avg Departure Delay

## Visualisations
- Donut chart — Overall passenger satisfaction (56.55% satisfied)
- Bar chart — Satisfaction by passenger type (Business vs Personal)
- Column chart — Satisfaction by travel type (Business · Economy · Economy Plus)
- Grouped bar chart — Passenger info by age & class
- Stacked bar chart — Passenger choice by flight leg

## Data Quality Steps
- Identified and resolved null values, blank fields, and duplicate records in Excel
- Cross-validated all KPI metrics against raw data before final publish
- Rebuilt data connection after cleaning to ensure accurate visualisations

## Dataset
Airline Quality Ratings — [Kaggle](https://www.kaggle.com)
